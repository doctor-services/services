# All doctor services repository
APIs and back-end projects
# Project structure
Put this project in to `$GOPATH/src/github.com/doctor-services/services`
# Package management
This project uses vndr. You should install `go get github.com/LK4D4/vndr` and but it into your `$GOAPTH`.
- Change current directory in to `$GOPATH/src/github.com/doctor-services/services`
- Run: `vndr` (Make sure you can exec vndr by putting it in your $PATH )
